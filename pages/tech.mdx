# Hitch Technology Stack

## Overview

Hitch is a web application that connects Northwestern students who need rides to and from airports, events, and other locations with students. Hitch is a full-stack web application that is built with the following technologies: 

- [TypeScript](https://www.typescriptlang.org/)
- [React](https://react.dev/)
- [TailwindCSS](https://tailwindcss.com/)
- [Python](https://www.python.org/)
- [Flask](https://flask.palletsprojects.com/en/3.0.x/)
- [PostgresSQL](https://www.postgresql.org/)
- [Render](https://dashboard.render.com/)
- [Firebase](https://firebase.google.com/)
