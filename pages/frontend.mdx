# Hitch Frontend Documentation 

In this section, we will cover the frontend of the Hitch application. The frontend is built using `React`, `TypeScript`, and `TailwindCSS`. We also used the componnt library `shadcn/ui` for some of the components.

## Folder Structure

The frontend folder structure is as follows:

```
frontend
├── public
├── src
│   ├── components
│       ├── ... // All the components used in the application
│   ├── ui
│       ├── ... // All the components from the shadcn/ui library
│   ├── utils
│       ├── ... // All the utility functions used in the application
│   ├── App.tsx
